# AI-NOW-Excel-project-D-Lite-Electronics
## Description
> This Excel work on D-Lite Electronics review case study aims to Extract, Clean, Analysis the company data to generate Insights that shows profit gained from sales in segment. Thereby generating insights that solves potential company problems. Key features are: Clean Data, Pivot tables and Dashboard report that support business decision making.
> Case Study by Morrison (Analyst).
## Project Overview
> This case study analyses D-lte Raw Inconsistent data,  cleans the data the generate insights from revenue, region, unit sold, category and sales representative to support business decision making.
## Data Content After Cleaning
### Rows: were 350 in Total.
 ### Columns: were 8 in Total.

## Tool Used
> Microsoft Excel (Excel functions, Pivot Tables & charts).

## Data Cleaning Actions
> Some data cleaning steps were applied to make the dataset proper for analysis without errors.

## Steps
### Date Column: cleaned using substitute and text.
### Region Column: cleaned using PROPER, TRIM, IF and NA.
### Product Column: cleaned using filtering and flash-fill
### Category Column: cleaned using filtering and flash-fill
### Sales Rep Column: cleaned using filtering and flash-fill
### Customer Column: cleaned using filtering and flash-fill
### Helper columns: calculated columns was created to aid analysis (Total Revenue)
